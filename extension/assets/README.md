# Extension Assets

## Icon Files Needed

The extension requires icon files in the following sizes:
- `icon-16.png` (16x16 pixels)
- `icon-48.png` (48x48 pixels)
- `icon-128.png` (128x128 pixels)

## Creating Icons

You can use any food/search related icon. Recommended sources:
- Create custom SVG icons
- Use icon generators like https://realfavicongenerator.net/
- Design with tools like Figma, Canva, or Adobe Illustrator

## Temporary Workaround

For quick testing, you can:
1. Use any 128x128 PNG image and resize it
2. Or temporarily remove icon references from manifest.json

## Recommended Icon Design

Theme: Food search / magnifying glass with food emoji
Colors: Green (#52b788) to match the OFF brand
Style: Simple, modern, flat design
